YASIR RAHIM Portfolio

Files:
- index.html
- style.css
- script.js

How to use:
1) Put all three files in the same folder.
2) Add your image in the same folder and rename it to:
   yasir.jpg
3) Open index.html in a browser.

Notes:
- No backend needed.
- Works on mobile and desktop.
- If yasir.jpg is missing, the page shows a fallback card automatically.
